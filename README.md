It helps you browse safely, encrypts your traffic, and ensures anonymity across networks.

## üîí Security Notice
When running `vpn.exe`, Windows may show a warning such as **"Unknown Publisher"** or **"Run Anyway."**  
This happens because the application is not signed with a commercial code-signing certificate.  
Code-signing certificates require licensing fees, which are not yet implemented for this project.
